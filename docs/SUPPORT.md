### üêõ Bug Reports

If you've found a bug, please [open an issue](https://github.com/juliuspleunes4/RenderSchema/issues/new) with:

- Clear description of the problem
- Steps to reproduce
- Expected vs actual behavior
- Python version and OS
- RenderSchema version (`pip show renderschema`)
- Minimal code example that reproduces the issue

### üí° Feature Requests

Have an idea for a new feature? [Open an issue](https://github.com/juliuspleunes4/RenderSchema/issues/new) describing:

- The problem you're trying to solve
- Your proposed solution
- Any alternatives you've considered
- Examples of how the feature would be used

### ‚ùì Questions

For questions about using RenderSchema:

1. **Check existing issues**: Someone may have already asked the same question
2. **Stack Overflow**: Tag your question with `renderschema` and `python`
3. **GitHub Discussions**: Start a discussion in our [repository](https://github.com/juliuspleunes4/RenderSchema/discussions)

### üìß Direct Contact

For security vulnerabilities or private concerns, email **jjgpleunes@gmail.com**.

**Please do not email for general support questions** - use GitHub issues/discussions instead so others can benefit from the answers.

## Response Times

- **Bug Reports**: We aim to respond within 48 hours
- **Feature Requests**: Reviewed weekly, prioritized based on community interest
- **Security Issues**: Addressed within 48 hours (see [SECURITY.md](SECURITY.md))
- **Pull Requests**: Reviewed within 3-5 business days
